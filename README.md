# VS3 Staff Console

Build 1 skeleton for Vintage Season 3 staff management.

## Included
- Next.js + TypeScript setup
- Sidebar admin layout
- Placeholder pages for Dashboard, Nodes, Factions, Weekly Tick, Instability Events, Staff Logs, Settings
- Supabase environment placeholders

## Run Locally
```bash
npm install
npm run dev
```

Open http://localhost:3000
